Web App Development - Project Milestone Report

Submitted for: CS 410 - Software Engineering

Project Overview

The goal of this project is to develop a fully functional web application that allows users to track their personal fitness goals. The project will include user authentication, workout tracking, and goal-setting features.

Milestone 1: Initial Setup & Design

In this milestone, we focused on setting up the basic project structure and designing the user interface. This included creating wireframes for the app's main screens and setting up a repository on GitHub.

  • Project repository created
  • Wireframes designed and approved
  • Initial front-end setup completed
Milestone 2: Authentication & User Management

This milestone focuses on implementing user authentication and allowing users to create accounts, log in, and manage their profile.

  • User authentication system implemented
  • Database setup for user data (to be completed by next week)
  • User profile page (currently in progress)
Milestone 3: Workout Tracking

The next milestone will focus on developing the workout tracking feature, which will allow users to log their exercise activities and track progress.

  • Implement tracking of exercises
  • Create workout history page
  • Testing and debugging phase
